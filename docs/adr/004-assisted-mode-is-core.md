# ADR 004: Treat assisted mode as a core pathway

## Status

Accepted

## Date

2026-04-14

## Context

A significant share of users may complete the screener with help from NGO staff, volunteers, union representatives, family members, or community workers.

## Decision

Assisted completion is a first-class pathway and will be supported in version 1.

## Consequences

- the product needs an explicit completion-mode choice
- copy and summaries must make sense for both self-serve and assisted use
- user research must include assisted sessions

## Review trigger

Review if research shows assisted mode is rarely used or if assisted and self-serve flows need to diverge materially.
